Create **local support tickets** (stored on your device only). They are not transmitted automatically. Use tickets to draft details, then email us if you need a human response.

| Screen | Contents |

| Screen | Contents |

### Core workflows

| Task | Where to go |
|------|-------------|
| Match one contact | Contacts → contact → Find Photo for This Contact |
| Save GitHub/LinkedIn handle | Contact → Manage Connections |
| Review suggestions | Review tab |
| Sync many contacts | Contacts → Sync Social Photos |
| Sign in to platforms | Settings → Account Sign-In |
| Import exports / photo folders | Settings → Import from Files |
| How to get photos (in app) | Settings → How to Get Photos |
| Save photo from Safari | Contact → Open in Safari → save → Choose Photo from Library |

## Before you contact support

1. Confirm **Contacts permission**: Settings → Privacy & Security → Contacts (iOS Settings).  
2. For matching issues: run **Photo Discovery Diagnostics → Run All Discovery Tests** and note failures.  
3. For billing/refunds: see [Refunds & billing guide](https://www.orangejuiceapplications.com/contact-profile-picture-sync/guides/refunds) or use Apple’s Report a Problem; this App is a **one-time paid download**, not a subscription.  
4. For data deletion: use **Privacy Dashboard → Delete All App Data** if instructed; note this does not delete entire Contacts.

- Requires **iOS** compatible with the current App Store build.

- **Photo library** permission only when uploading.  
- **Face ID / Touch ID** only if you enable biometric lock.
