Two paths in the app

Choose Account Sign-In or Import from Files — switch tabs to see each flow.

Path A — Account Sign-In

  1. Open Settings
  2. Tap Account Sign-In
  3. Choose a platform → Sign In
  4. Enable it under Photo Sources
  5. Contacts → Sync Social Photos
  6. Approve in the Review tab

Social sign-in guide →

Path B — Import from Files

  1. Open Settings
  2. Tap Import from Files
  3. Choose import type (folder, LinkedIn CSV, etc.)
  4. Tap Choose File…
  5. Open Review Queue after import
  6. Or: contact → Share Photo from Browser → Share → FaceMatch
